# Task Management Tools

http://cool.ssnetwork.io:52489/   (モバイル用レイアウトが未実装のため、PCでの閲覧を推奨しています。)

Flask を使用して作成した **シンプルなタスク管理 Web アプリケーション**です。  
ユーザー登録・ログイン機能を備え、タスクの追加・完了管理・削除を行うことができます。

タスクデータおよびユーザー情報は **FTPサーバー上のテキストファイルとして保存**されます。  
また、簡易的な **グローバルチャット機能**も実装されています。

※通信から保存まで平文で行っているため、セキュリティ的にはかなり問題があります。これはあくまでも技術デモです。

---

# 機能

## ユーザー管理

- ユーザー登録
- ログイン / ログアウト
- セッション管理

ユーザー情報は以下のファイルに保存されます。

```
userlist/list.txt
```

フォーマット

```
username,password
```

---

## タスク管理

ログインしたユーザーごとにタスクを管理できます。

### タスク操作

- タスク追加
- タスク完了状態の切り替え
- タスク削除
- タスク一覧表示

各ユーザーのタスクは以下に保存されます。

```
tasks/<username>.txt
```

フォーマット

```
done,name,date,time,detail
```

例

```
0,課題提出,2026-03-10,23:59,レポートを提出
1,ミーティング,2026-03-11,14:00,チーム会議
```

|項目|説明|
|---|---|
|done|完了フラグ (0 / 1)|
|name|タスク名|
|date|日付|
|time|時間|
|detail|詳細|

---

## チャット機能

ログインユーザー同士で簡単なチャットを行うことができます。

保存ファイル

```
chat/global.txt
```

フォーマット

```
username,time,message
```

例

```
mizuki,10:32,おはよう
tanaka,10:33,こんにちは
```

表示は **直近50件**まで取得されます。

# 必要環境

- Python 3.8 以上
- Flask
- FTP サーバー

インストール

```bash
pip install flask
```

---

# 設定

`app.py` の以下を設定してください。

```python
FTP_HOST = ''
FTP_USER = ''
FTP_PASS = ''
```

FTPサーバーには以下のディレクトリを作成する必要があります。

```
userlist/
tasks/
chat/
```

初期ファイル

```
userlist/list.txt
```

---

# 起動方法

```bash
python app.py
```

ブラウザでアクセス

```
http://localhost:5000
```
